Режими резервного копіювання: Із застосунку можна створити резервну копію лише її apk, лише її дані або обидва. Програму, не встановлену на цю мить, можна відновити лише за наявності файлу .apk. Три додаткові типи даних підтримуються додатково до основних даних: захищені пристроєм дані, зовнішні дані та дані obb.

Видалити резервну копію: Видаляє файли резервної копії для певного екземпляра резервної копії. "Видалити резервні копії" видалить усі резервні копії вибраного застосунка.

Видалення: Дещо агресивніший, ніж звичайне видалення. На застосунок до звичайного видалення за допомогою команд Android, видалення з OAndBackupX видаляє файли, які застосунок могла залишити в /data/app-lib/.

Видалити резервну копію: Видаляє файли резервної копії для певного екземпляра резервної копії. "Видалити резервні копії" видалить усі резервні копії вибраного додатка.

Видалення: Дещо агресивніший, ніж звичайне видалення. На додаток до звичайного видалення за допомогою команд Android, видалення з OAndBackupX видаляє файли, які програма могла залишити в /data/app-lib/.

Multiple users: Multi-user is still somewhat experimental but should work. When restoring in a multi-user scenario, `pm install -r $apk` gets called and subsequently the app is disabled for every user who has the app listed in /data/system/user/$user/package-restrictions.xml (unless the app is listed as enabled="1"). <br /> This can create problems for users installing the same app at some later point, but is necessary to prevent the app from being installed to all users at the same time. The context menu has an option to enable or disable apps which can be used if other users become unable to use a specific app due to disabling on restore. <br /> Enabling and disabling only works after an initial install (not necessarily from OAndBackupx) or restore of the app.

System-less apps: Backup and restore of system-less apps installed as Magisk modules is not supported and trying it could lead to system failure and forcing you to format and flash it anew.

Навігація . На панелі навігації є "Налаштування", "Головна", "Пакетне резервне копіювання", "Пакетне відновлення" та "Графік резервних копій" (у такому порядку). Кількість програм, оновлених після останнього резервного копіювання, буде представлено як сповіщення на кнопці "Головна".
